## Usage

Create realistic NFL mock drafts with SundayScouts.com

- Control one or more teams.
- Pick college prospects for your teams.
- Simulate draft picks for the other teams.
- Simulate 1-7 rounds.
- Download an image summary of your draft.
- Trade Draft Picks
- View Prospect Scouting Profiles
- Filter Prospects by College, Position
- Compare Prospects from the same position.

For more examples, please refer to the Documentation
